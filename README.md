# House Price Prediction Using Machine Learning

This project is a Machine Learning-based House Price Prediction system developed using Python, Scikit-learn, Flask, and web technologies.

The application predicts house prices based on multiple housing features using a trained Random Forest Regression model.

---

# Technologies Used

- Python
- Pandas
- NumPy
- Scikit-learn
- Flask
- HTML/CSS
- Joblib
- Jupyter Notebook

# Files Description

- `main.py`
  Main Flask application file.

- `data.csv`
  Dataset used for training and prediction.

- `real_estate_price_prediction.ipynb`
  Jupyter Notebook containing machine learning model training and preprocessing.

- `real_estate_price_predictor.pkl`
  Saved trained machine learning model.

- `templates/index.html`
  Frontend webpage for user interaction.

- `static/`
  Contains images and static assets.
